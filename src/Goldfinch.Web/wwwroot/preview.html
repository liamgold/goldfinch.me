$ favicon set — delivered

Seven files, drop the whole lot straight into wwwroot/ (root of the site — site.webmanifest expects the PNGs at /). SVG is the primary, favicon.ico bundles 16/32/48 for legacy clients, apple-touch-icon is flat (no alpha, no rounded corners — iOS handles both).

Assets

vector
primary, any DPI
favicon.ico
16 / 32 / 48
multi-res legacy

PNG fallback
apple-touch-icon.png
180 × 180, flat
iOS home screen
manifest-192.png
192 × 192

maskable, splash

Markup for <head>

<!-- Favicons -->
<link rel="icon" href="/favicon.ico" sizes="48x48">
<link rel="icon" href="/favicon.svg" type="image/svg+xml">
<link rel="icon" href="/favicon-96x96.png" type="image/png" sizes="96x96">
<link rel="apple-touch-icon" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">
<meta name="theme-color" content="#0c0d0f">
